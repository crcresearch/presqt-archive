September 17 and 18, 2018

Monday Sept 17, 2018 Noon to 5

Noon-1pm Lunch

Opening Remarks during lunch, including Timeline Info [John Wang]

1-1:30 Articulate Workshop Goals / Present High Level Design / How Sprints fit within Timeline [Sandra Gesing]

  • Pamfilos Fokianos - Tools to Improve Preservation and Re-use of Research Results at CERN

  • Elliot Metsger - PASS/Fedora Demo

Break: 2:45-3

3-4pm Breakout Groups

Interoperability with Repositories and Preservation Services [Rick Johnson]: (OSF, Fedora,Hubzero, SHARE)
  • Maturity of API and web services
  • Which participant systems & tools can interoperate

  • Maturity of API and web services
  • Which participant systems & tools can interoperate

  • Maturity of API and web services
  • Which participant systems & tools can interoperate
