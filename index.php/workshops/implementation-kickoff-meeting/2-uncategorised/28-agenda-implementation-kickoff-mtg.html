Kickoff Mtg Agenda

September 17 and 18, 2018

Monday Sept 17, 2018 Noon to 5

Noon-1pm Lunch

Opening Remarks during lunch, including Timeline Info [John Wang]

1-1:30 Articulate Workshop Goals / Present High Level Design / How Sprints fit within Timeline [Sandra Gesing]

1:30-2:45 Brief Presentations [Facilitated by Rick Johnson/Natalie Meyers]

  • Pamfilos Fokianos - Tools to Improve Preservation and Re-use of Research Results at CERN
  • Nathan Denny - HUBzero Introduction
  • Elliot Metsger - PASS/Fedora Demo
  • David Minor / Rick Johnson - SHARE: Gathering and Surfacing Research Activity
  • Craig Willis - NDS Labs Workbench: A Scalable Platform for Research Data Access, Education and Training
  • Vicky Steeves - Reproducing & Archiving Scholarship with ReproZip

Break: 2:45-3

3-4pm Breakout Groups

Interoperability with Repositories and Preservation Services [Rick Johnson]: (OSF, Fedora,Hubzero, SHARE)
  • Maturity of API and web services
  • Which participant systems & tools can interoperate
Interoperability with Research Data Systems [Sandra Gesign] (HubZero, OSF, Jupyter, NDS, ReproServer)
  • Maturity of API and web services
  • Which participant systems & tools can interoperate
Interoperability w/Software [Natalie Meyers/John Wang] (EaaS, NDS, CodeOcean, WholeTale, Reprozip)
  • Maturity of API and web services
  • Which participant systems & tools can interoperate

4-4:50 Report Back [Natalie Meyers]

  • Each group summarize API maturity
  • Each group summarize identified interoperability opportunities
  • Identify alignments for Tuesday’s breakout groups

4:50-5 Wrap Up

6pm - Social Gathering @ Danny Boy’s

Tuesday 18th: 9 AM to 3:30 PM

8:30-9am Breakfast

9-9:15 Recap and Day’s Goals [Sandra Gesing]

9:15-10:45 AM Features Development Breakout Groups

  • Fixity Service Requirements and their support of this area [Natalie Meyers]
  • Keyword Assignment Requirements and their support of this area [Rick Johnson]
  • Preservation Quality Service Requirements and their support of this area [Sandra Gesing/John Wang]

10:45-11: Break

11-12: Presentations & Open Discussion [ Facilitated by Natalie Meyers/Rick Johnson]

  • Craig Willis - The WholeTale: Merging Science and Cyberinfrastructure Pathways
  • April Clyburne-Sherin - Code Ocean
  • Jeffrey Spies- OSF API
  • Susan Borda - Data Curation Network Update
  • Euan Cochrane - Emulation as Service Infrastructure - EaaSI
  • Open Discussion

12-1 Lunch

1-2:30 Talks/Open Discussion [Sandra Gesing]
Systems’ Focus

  • Work Planning for tools/services that will interoperate with mature systems
  • Work planning for APIs/target systems that need to be improved before they are ready for interoperability dev & testing
  • Assignment of expectations
    • Design
    • Coding (code repository, code review, merges)
    • Testing
    • Sprints - place within overall timeline

2:30-3pm Communication Overview [Natalie Meyers & Miranda Van Nevel]

  • Priorities
  • Sprints
  • Documentation
  • Communication methods, frequency & audiences

3pm Closing Remarks [John Wang]
